Topic via Dialogue

Scene: You just typed claude "fix this bug" in the terminal...

N
Newbie

So when I hit Enter, what actually happens? Does the AI just... read my code?

S
Senior

Not quite! Think of it as a conversation loop. query.ts sends your message to the Claude API, gets back a response, and then checks: "Did Claude ask to use any tools?" If yes, it executes the tools and loops back.

N
Newbie

Oh, so it's like a waiter going back and forth between the customer and the kitchen?

S
Senior

Exactly! And the "menu" is the tool registry in tools.ts — each tool has its own rules about who can order it.

Think About It

Why does this module use lazy loading instead of static imports?

Reference answer: Lazy loading breaks circular dependency chains between modules, which is critical because tool.ts and query.ts would otherwise import each other at module parse time.
